Ruby on Rails 应用企业级管理框架

Active Admin 是一款用于生成 Ruby on Rails 管理界面的插件。它通过高度概括总结常见商业应用程序的逻辑模式，化繁为简，使开发者只需使用很少的代码就能生成漂亮优雅的界面。

为真正用户设计的漂亮界面。

全局导航

定制全局导航可以为您的商务活动创建可重用的管理界面。

分类

利用 Scope 可以通过不同角度和方法创建更合适的数据资源导航和报告。

索引方式

索引有多种方式。默认使用表格（Table）方式，但是 Active Admin 也支持网格（Grid）、块（ Block）和博客（Blog）方式。

API & 下载

在 Active Admin 注册的每个资源都提供 JSON、XML 和 CSV 格式的数据下载。您可以根据需要定制输出。

用户认证

利用绑定的 Devise 配置或者通过系统提供的钩子函数扩展您自己的认证系统。

操作项

在每个页面的操作项栏目可以添加按钮、链接或者其它内容。

过滤器

用户可以通过搜索词、文本框、日期和数字值等对数据进行过滤。

边栏

使用 Active Admin 内置的简单的 DSL 可以构建边栏。

Active Admin 的界面设计伊始就是面向非技术用户的。所以开发人员构建高质量的时用于客户的界面就很容易。

面向开发者的高质量的 DSL。

利用 DSL，始于一行代码，定制整个界面。 

# app/admin/products.rb
ActiveAdmin.register Product do
  # Create sections on the index screen
  scope :all, default: true
  scope :available
  scope :drafts

  # Filterable attributes on the index screen
  filter :title
  filter :author, as: :select, collection: ->{ Product.authors }
  filter :price
  filter :created_at

  # Customize columns displayed on the index screen in the table
  index do
    column :title
    column "Price", sortable: :price do |product|
      number_to_currency product.price
    end
    actions
  end
end
